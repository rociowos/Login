Keys You Need to Run the Program

1. Obtain the Weather API Key
    1. Create an account at OpenWeatherMap.
    2. Copy the provided API key.
    3. Create a .env file in the root directory of your project with the following content:

        WEATHER_API_KEY="your_api_key"


2. Generate the Secret Key (SECRET_KEY)
    1. Run the following code in a Python script to generate a random secret key:
       
        import os
        import binascii

        secret_key = binascii.hexlify(os.urandom(24)).decode()
        print(secret_key)
    
    2. Copy the random string that you get as output. This will be your SECRET_KEY.



3. Create the Admin Password Hash
    1. Run the following code in a Python script to generate the hash for the admin password:
         
        import bcrypt
        password = 'admin'  # Admin password

        # Generate the password hash
        hashed = bcrypt.hashpw(password.encode('utf-8'), bcrypt.gensalt())

        # Print the hash to the console
        print(hashed.decode('utf-8'))

    2. Copy the hash that you get as output. This will be your ADMIN_PASSWORD_HASH.


4. Create the Config.py File
    1. Create a file named Config.py in the root directory of your project.
    2. The file should look like this, replacing 'the_random_string' and 'admin_hash' with the values generated in the previous steps:

        import os

        class Config:
            SECRET_KEY = 'the_random_string'  # Replace with the generated secret key
            SQLALCHEMY_DATABASE_URI = 'sqlite:///site.db'  # Or the URI for your database
            SQLALCHEMY_TRACK_MODIFICATIONS = False
            ADMIN_PASSWORD_HASH = 'admin_hash'  # Replace with the generated admin password hash
